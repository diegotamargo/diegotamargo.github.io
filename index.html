01RAG SYSTEM

NormaINGECA

A search tool that lets engineers ask questions about compliance and regulatory documents in plain language instead of digging through PDFs. It pairs BM25 keyword search with vector retrieval and merges the two using Reciprocal Rank Fusion. The embedding model runs locally, picked so it works on ordinary office machines. It's now installed on more than 50 workstations as a standalone Windows app.

02GEOSPATIAL / CV

Orthophoto Vectorization Pipeline

A system I'm designing to turn Spain's PNOA aerial imagery into clean, georeferenced vector layers for three things: buildings, roadway, and sidewalks. Rather than one model for everything, each class gets its own approach — building polygons cross-checked against the cadastre, the roadway traced as a graph so it survives cars and shadows, and sidewalks segmented properly since you can't just infer them from the space between road and facade. LiDAR and the infrared band help with occlusion and telling ground from rooftops. It's still in development and not deployed yet; the segmentation models need more GPU memory than I have on hand, so for now the focus is on the design and the data pipeline.

03CAD TOOLING

DXF Normalization Tool

A desktop tool that tidies up messy CAD layer structures, folding them down to a standard set. The rules live in a YAML file rather than in the code, so they're easy to change. It runs as a Windows app with a system tray icon, and I've prepared it for release under an MIT license.
